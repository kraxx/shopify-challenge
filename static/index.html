Shopping API

This API implements GraphQL

Detailed instructions on how to use this API: here

Here's the full query string to get everything in our database:

/graphql?query={shop{id,name,products{id,shop_id,name,value,quantity,line_items{id,product_id,order_id,quantity}}orders{id,value,line_items{id,product_id,order_id,value}}}}

Hit the /reseed endpoint if things go awry (on success, you'll be redirected back here)

You might want to use a JSON prettifier such as https://jsonformatter.curiousconcept.com/
